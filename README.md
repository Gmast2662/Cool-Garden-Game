# Garden Game

This is a cool garden game where you grow a garden and can do some other stuff. This is made by me (and a little help from AI) and is a cool project.

## 🎮 How to Play

1. Click an empty dirt patch `🟫` to plant a sprout.
2. Click the sprout `🌱` to help it grow into a leafy plant `🌿`.
3. Click it one more time to harvest a fully grown sunflower `🌻`!
&nbsp;

You can clone this repository or play here: (later) on pages.

To get started, install the dependencies:
```bash
npm install
```
and start the site:
```bash
npm run dev
```
